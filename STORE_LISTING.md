Ready-to-paste texts for Chrome Web Store and Firefox AMO.

## Chrome Web Store

### Name (max 45 chars)
```
SmartShot — Full Page Screenshot
```

### Short description (max 132 chars)
```
Capture full-page screenshots to JPG/PNG. Save to file or clipboard. 100% local — no account, no cloud, no data stored.
```

### Detailed description (max 16,000 chars — paste as-is)
```
SmartShot captures entire web pages from top to bottom in one click — no stitching required on your part.

FEATURES
• Full-page capture — automatically scrolls and stitches the entire page into one image
• Select Area — drag to capture any region of the visible page
• Save to file — JPG or PNG saved directly to your Downloads folder, auto-named with domain and timestamp
• Copy to clipboard — instantly paste into Teams, Slack, Jira, Outlook, Notion or any app
• Save + Copy — both at the same time
• Countdown delay — set 1–10 seconds before capture starts, perfect for open menus and tooltips
• Preview tab — review the screenshot in a new tab before saving
• Capture history — your last 10 screenshots shown in the popup
• Keyboard shortcut — Alt+Shift+S for instant capture without opening the popup

PRIVACY — 100% LOCAL
SmartShot does not collect any data. It does not connect to any server. It requires no account, no sign-in, and no subscription. Everything happens on your computer. The source code is publicly available on GitHub.

PERMISSIONS EXPLAINED
• activeTab — to read the current page for capture
• scripting — to inject the capture script into the page
• downloads — to save screenshots to your computer
• storage — to save your settings locally

Open source · MIT licence · https://github.com/joncax/smartshot
```

### Category
`Productivity`

### Screenshots needed (1280×800 or 640×400)
1. Popup with Full Page mode and history visible
2. Area selection overlay on a real page
3. Settings page
4. Screenshot result saved in Downloads

## Privacy Policy (required by both stores)

Host this text at a public URL (GitHub Pages, Notion, etc.) and link it in the store listing.

```
SmartShot Privacy Policy

Last updated: March 2026

SmartShot does not collect, transmit, store or share any personal data or browsing data.

All operations (capture, save, clipboard, history) happen entirely on the user's device.
No data is sent to any external server.
No analytics or tracking of any kind is included.
The capture history is stored locally in the browser and never leaves the device.

This extension requires the following permissions:
- activeTab: to access the current page for capture
- scripting: to inject the capture script into the page
- downloads: to save screenshot files to the user's computer
- storage: to save user settings in the browser

For questions: https://github.com/joncax/smartshot/issues
```
